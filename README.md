# Neighbor_Joining_JC
This is a repository containing code I wrote to implement Jukes-Cantor distance and the Neighbor Joining phylogenetic Algorithm


Jukes_Cantor.py contains the functions to calculate hamming distance, and calculate the Jukes Cantor Distance between all taxa in a FASTA file.

Newick_Neighbor.py contains the functions used to perform neighbor joining, build a tree dictionary from a set of edges, and make a Newick Tree.

Run_Top_to_Bottom.py is where I ran this implementation on my Fasta Files as well as a few examples I used to debug.
